# MinTTY-Theme-Converter
Creates a theme for git bash on Windows given 19 colors.      The program takes in a filename as input and produces a file     that is to be placed in "C:/Program Files/Git/usr/share/mintty/themes"     The program uses the  16-Color ANSI Palette (used by CLI tools) + 3 colors      for ForegroundColour, BackgroundColour, CursorColour
